Display - elementning web sahifada qanday ko'rinishini belgilaydi

Display turlari

  • Block
  • Inline
  • Inline-Block
  • None

Block elementi ajdodiga nisbatan 100% kenglik va yangi qator talab qiladigan va CSS xossalarini ozida jamlay oladigan elementdir.

Inline elementi o'ziga yetarlicha joyni va yangi xatorni talab qilmaydigan, margin, padding xossalarini o'zida jamlamaydigan elementdir.

Inline-Block elementi o'ziga yetarlicha joy qiladi, yangi qator talab qilmaydi va margin, padding xossalarini o'zida jamlay oladigan elementdir.

None elementi web sahifada ko'rinmaslik xossasini qabul qiladi x
